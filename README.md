# Información

```
Aplicación web de prueba perteneciente al Curso Node.js.
Se busca utilizar express para servir una página en principio estática, pero con la 
utilización de Handlebars convertirla a una web dinámica.
Además sienta las bases para desplegar la aplicación en un hosting, utilizando variables de entorno 
para asignar el puerto de servidor y script para levantar la aplicación.
```
